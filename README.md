# Formulário Básico em HTML e CSS

##  Descrição

Este projeto consiste em um formulário básico desenvolvido com **HTML** e **CSS**. Ele foi criado com o objetivo de **fixar e revisar os conceitos fundamentais** dessas tecnologias durante meus estudos.

##  Objetivo

* Reforçar a estrutura básica do HTML
* Praticar estilização com CSS
* Entender a construção de formulários
* Aplicar boas práticas iniciais de organização do código

##  Tecnologias Utilizadas

* HTML5
* CSS3

##  Funcionalidades

* Campos de entrada de dados (input)
* Rótulos (labels)
* Botão de envio
* Estilização simples e organizada

##  Estrutura do Projeto

```
/formulario-basico
│── index.html
│── style.css
```

##  Como Executar o Projeto

1. Faça o download ou clone este repositório
2. Abra o arquivo `index.html` em qualquer navegador

##  Observações

Este projeto tem fins **educacionais** e não possui integração com backend ou banco de dados.

Projeto desenvolvido para fins de estudo e prática em HTML e CSS.
